Database Maintenance

Execute SQL queries for database maintenance

⚠️ Warning: This tool allows direct database manipulation. Use with caution and make sure you understand the queries before executing them. Always backup your data before running destructive operations.

Quick Actions:

UPDATE team_members SET photo = NULL WHERE photo IS NOT NULL;
SELECT COUNT(*) as total_members, COUNT(photo) as members_with_photos FROM team_members;
SELECT id, name, photo FROM team_members WHERE photo IS NOT NULL LIMIT 10;
UPDATE team_members SET photo = NULL WHERE photo LIKE '%/api/photos%';
